# Jotly
Веб-приложение для ведение личного дневника

## Структура проекта
- `docker-compose.yml` — описание контейнеров (веб-приложение и база данных)
- `Dockerfile` — инструкция по сборке Docker-образа Django-приложения
- `config/` — конфигурация проекта Django (settings, urls и т.д.)
- `diary/` — приложение для работы с дневниковыми записями
  - `templates/diary/` — HTML-шаблоны, связанные с записями
- `users/` — приложение для управления пользователями (регистрация, вход, профиль)
  - `templates/users/` — HTML-шаблоны, связанные с пользователями
- `static/` — статические файлы (CSS, JS, изображения)
- `requirements.txt` — зависимости проекта
- `README.md` — документация по установке и использованию проекта

## Основной функционал
- Регистрация и вход по Email
- CRUD-операции для записей
- Поиск записей по содержимому и заголовку
- Контейнеризация с Docker и PostgreSQL

## Установка с помощью Dooker

1. Клонируйте репозиторий на свое локальное утсройство
2. Создайте виртуальное окружение (опционально)	2. Используя .env.sample создайте свой .env файл с вашими параметрами
3. Установите зависимости Проверьте, что у вас установлен Docker
4. Запустите контейнеры в корневой папке проекта через команду команду sudo docker-compose up --build
   
## Инструкции к использованию	
Приложение будет доступно по адресу http://localhost:8000
